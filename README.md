Ext.ux.data.proxy.PagingMemory
==============================

Paging memory proxy, allows using in-memory dataset with paging grid.
Similar to Ext.ux.data.PagingMemoryProxy except that it's optimized for large datasets.

Tested with:
    - MSIE 6+
    - Chrome 6+
    - Firefox 3.6+
    - Opera 11
    - Safari 4+
